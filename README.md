AI-powered tool to:

## Setup

1. Clone repo or unzip
2. Create `.env` with keys:
   ```
   OPENAI_API_KEY=your_key_here
   ELEVENLABS_API_KEY=your_key_here
   ELEVEN_VOICE_ID=EXAVITQu4vr4xnSDxMaL
   ```
3. Install deps:
   ```bash
   pip install -r requirements.txt
   ```
4. Run:
   ```bash
   python app.py
   ```
